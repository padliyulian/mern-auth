# MERN Auth Boilerplate
- author : padliyulian@gmail.com

# Back End
- Node JS, Express
- mongoDB

# Front End
- React, Next JS (typescript)

# Feature
- login
- register
- forgot & reset password
- email notif
- Access Control List

# BE Setup
- docker-compose up -d
- yarn install
- yarn start

# FE Setup
- yarn install
- yarn dev
